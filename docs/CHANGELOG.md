# 📝 Changelog do Crystal Legacy

Todas as mudanças importantes do projeto serão documentadas neste arquivo.

---

## [0.1 Alpha] — Fundação inicial

### Adicionado

- Criação do repositório CrystalLegacy.
- Criação da estrutura inicial de pastas.
- Criação do README principal.
- Criação da Constituição do projeto.
- Criação do Roadmap inicial.
- Criação do documento de Arquitetura.
- Criação do Guia de Design.
- Definição inicial da filosofia do projeto.
- Definição inicial da identidade visual.
- Definição de que o projeto será inicialmente em Português (Brasil).
- Definição do lema: “Forje o seu próprio legado.”

### Decisões

- O projeto será baseado no Arch Linux.
- O ambiente gráfico inicial será KDE Plasma.
- O projeto priorizará leveza, clareza e aprendizado opcional.
- O Zen Browser será usado como navegador principal no ambiente de desenvolvimento.
- O Cavaleiro será um assistente opcional e ficará para uma fase futura.
- O projeto não copiará elementos protegidos de outras obras.

---

## Regras do Changelog

Cada nova versão deverá registrar:

- O que foi adicionado.
- O que foi alterado.
- O que foi removido.
- O que foi corrigido.
- Decisões importantes tomadas no projeto.
